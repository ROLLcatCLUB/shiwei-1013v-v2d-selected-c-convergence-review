# V2D Selected C Convergence Notes

## 已收敛项

### 1. 纸面网格降噪

V2D 继续保留 C 方向的纸面感，但把舞台和正文区域网格调淡：

- 舞台网格透明度降到 `0.026 / 0.018`。
- 正文纸面网格透明度降到 `0.022 / 0.018`。
- 正文纸面保持暖白底，不再增加绿色重量。

### 2. 左侧备课本目录降权

左侧仍是备课本目录，但不抢正文：

- 整体 opacity 降到约 `0.74`。
- 鼠标悬停或聚焦时提升到约 `0.94`。
- 非当前单元/课例降到约 `0.52`。
- 当前课例保持清晰。

### 3. 右栏改成教师旁注

右栏只表达当前教案会影响什么：

```text
这份教案将影响
1. 课堂大屏：生成 2 页观察引导
2. 学习单：生成 1 个找渐变任务
3. 评价点：记录学生能否说出颜色变化方向
```

右栏动作只保留：

```text
预览课堂材料
查看全部
```

### 4. 底部小教条保持系统级入口

底部小教条保留为一级壳的系统输入入口：

- 高度维持在 56-64px 区间。
- 输入框保持居中。
- 不覆盖备课本底部内沿。

### 5. 工程词隐藏

教师默认可见/可访问文本不得出现：

```text
affected_fields
downstream_dirty_fields
preview_delta
static only
preview only
preview_only
formal_apply
小教判断区
```

保留在静态 JSON 或脚本里的历史/审计数据会打 `data-audit-only="true"` 和 `aria-hidden="true"`。
